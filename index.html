spalotus

A basic starting point for scss and js

This baseline is intended as a starting point for structuring scss in an app. It includes;

  • The new css reset (2022)
  • Google Material Symbols
  • A simple javascript toggle for dark / light mode based on Christian Oliff's dark mode switch.
    • Not working as intended. Always prioritises dark if device preference set to dark. Local storage should be the priority. Needs a re-write.
    • Intended behaviour is no local storage saved unless toggle clicked.
    • Idea: auto toggle mode if system preference changes since toggle clicked.

Works well for very quick styling when used in conjunction with Codekit.
